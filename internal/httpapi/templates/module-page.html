Install

Direct Tarball Install

Download the selected release tarball from this service and install it directly.

shell 
curl -L {{.PublicBaseURL}}{{.DownloadPath}} -o {{.ModuleInstallID}}-{{.Release.Version}}.tar.gz
puppet module install ./{{.ModuleInstallID}}-{{.Release.Version}}.tar.gz --ignore-dependencies

Use This Forge As Repository

Point Puppet to this service as a custom module repository for the selected module version.

shell 
puppet module install {{.ModuleInstallID}} --version {{.Release.Version}} \
  --module_repository {{.PublicBaseURL}}

Puppetfile With This Forge

Configure your Puppetfile to use this service as the Forge source and pin the required module version.

Puppetfile 
forge "{{.PublicBaseURL}}"

mod "{{.ModuleInstallID}}", "{{.Release.Version}}"

Private Forge Access For r10k

If your forge is protected, configure r10k Forge access with a read token and keep the module version pinned in Puppetfile.

Hiera 
forge:
  baseurl: '{{.PublicBaseURL}}'
  authorization_token: '{{.ReadTokenHint}}'
Puppetfile 
forge "{{.PublicBaseURL}}"

mod "{{.ModuleInstallID}}", "{{.Release.Version}}"
{{end}}
